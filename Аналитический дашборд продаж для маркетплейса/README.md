# Анализ продаж маркетплейса
__________________________________

## Описание проекта
__________________________________
Аналитический дашборд, разработанный в Apache Superset, содержащий 4 вкладки: 
-	«О дашборде»
-	«Основные показатели»
-	«Детализация по покупателям и продавцам»
-	«Детализация по оценке качества и отзывам»

Для реализации проекта используются 2 датасета: "Обогащенные заказы" (основной для построения графиков по продажам, категориям товаров, активности продавцов и покупателей) и "Пользователи и их активность" (для анализа регистраций, активности пользователей, а также для построения графиков Retention и активности по времени)

## Цели дашборда
__________________________________
Данный дашборд предназначен для комплексного анализа эффективности маркетплейса. С его помощью вы можете отслеживать ключевые метрики продаж, анализировать поведение покупателей и продавцов, а также оценивать качество сервиса.

## Основные направления анализа
__________________________________
Анализ эффективности продаж
Мониторинг динамики выручки (GMV) и объемов продаж
Анализ среднего чека и размера корзины
Оптимизация ассортимента
Определение наиболее прибыльных товаров и категорий
Отслеживание конверсии и удержания пользователей
Расчет LTV и ARPU для оценки ценности клиентов
Мониторинг ежедневной активности (DAU)
Оценка качества сервиса
Мониторинг удовлетворенности клиентов

## Инструменты
__________________________________
SQL, Apache Superset

## Результаты
__________________________________
Разработан интерактивный дашборд с подробной документацией, который удобно использовать для оценки продаж и внесения корректировок в ассортимент товаров
